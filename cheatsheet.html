SELECT
- Syntax: select column_name from table_name;
WHERE
- Syntax: select column_name from table_name where condition;
AND & OR
- Syntax: select column_name from table_name where condition_1 or condition_2;
ORDER BY
- Syntax: select column_name from table_name where condition_1 order by column_name desc;
- Use desc for descending and asc for ascending
LIKE
- Syntax: select column_name from table_name where column_name like pattern;
INNER JOIN
- Syntax: select column_name from table_1 inner join table_2 on table_1.column_name=table_2.column_name
WILDCARD CHARACTERS
% - A substitute for zero or more characters
_ - A substitute for one character
[charlist] - Sets a range of characters to match
[!charlist] or [^charlist] - Sets a range of characters to NOT match
